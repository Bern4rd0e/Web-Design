Design Centrado no Usuário

  • Tem como ideia principal centralizar o usuário no núcleo decisivo e o manter durante todas as etapas do projeto, aumentando portanto, a chance de uma maior adesão do resultado final com o público alvo;
  • Metodologias criadas para ajudar profissionais a desenvolver interfaces para atender às necessidades dos usuários.
Empatia Para que haja dedicação às necessidades do usuário.
Simbiose Colaborativa entre os clientes e os projetistas durante o desenvolvimento.
Otimismo Para criar um cenário de legitimidade na mudança.
Resiliência Para um melhor aprendizado através das tentativas e experiências.
